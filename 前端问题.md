
### 2.3 登录后跳转逻辑

```javascript
const { token, user } = await login(email, password);

// 保存 Token
localStorage.setItem('token', token);
localStorage.setItem('user', JSON.stringify(user));

// 根据角色跳转
if (user.role === 'admin') {
  window.location.href = '/pages/admin/dashboard';
} else {
  window.location.href = '/home';
}
```

## 九、错误处理

### 9.1 HTTP 状态码

| 状态码 | 含义 | 处理建议 |
|--------|------|----------|
| 200 | 成功 | 正常处理响应 |
| 201 | 创建成功 | 正常处理响应 |
| 400 | 参数错误 | 提示用户检查输入 |
| 401 | 未授权 | 清除 Token，跳转登录页 |
| 403 | 禁止访问 | 非管理员用户，禁止访问管理员接口 |
| 404 | 资源不存在 | 提示用户资源不存在 |
| 409 | 冲突 | 根据错误信息提示（如版本有关联卡牌无法删除） |
| 500 | 服务器错误 | 提示用户稍后重试 |

### 9.2 统一错误处理

```javascript
async function apiRequest(method, path, data) {
  try {
    const result = await api.request(method, path, data);
    return { success: true, data: result };
  } catch (error) {
    if (error.message.includes('Unauthorized')) {
      // Token 过期，跳转登录
      localStorage.removeItem('token');
      localStorage.removeItem('user');
      window.location.href = '/login';
      return { success: false, redirect: '/login' };
    }

    if (error.message.includes('Forbidden')) {
      // 非管理员，禁止访问
      window.location.href = '/home';
      return { success: false, error: '无权限访问' };
    }

    return { success: false, error: error.message };
  }
}
```

### 10.2 权限检查

```javascript
// 在管理员页面加载时检查权限
function checkAdminAccess() {
  const user = JSON.parse(localStorage.getItem('user') || '{}');

  if (user.role !== 'admin') {
    window.location.href = '/home';
    return false;
  }
  return true;
}

// 页面加载时调用
onMounted(() => {
  checkAdminAccess();
});
```

### 10.3 游戏版本管理页面

#### 功能需求

1. **列表展示**：展示所有游戏版本
   - 显示字段：versionId、name、lang、logo
   - 支持分页（可选）

2. **新增版本**
   - 表单字段：name（必填）、lang（选填）
   - 调用 `POST /v1/api/admin/versions`

3. **编辑版本**
   - 可编辑：name、lang、logo
   - 调用 `PUT /v1/api/admin/versions/:versionId`

4. **删除版本**
   - 调用 `DELETE /v1/api/admin/versions/:versionId`
   - 有关联卡牌时提示：`Cannot delete version with associated cards`

5. **上传 Logo**
   - 支持 png/jpg/webp，建议 ≤ 2MB
   - 调用 `POST /v1/api/admin/versions/:versionId/logo`

#### UI 建议


## 十一、常见问题

### Q1: Token 过期怎么办？

```javascript
// 统一在请求拦截器中处理
axios.interceptors.response.use(
  response => response,
  error => {
    if (error.response?.status === 401) {
      localStorage.removeItem('token');
      localStorage.removeItem('user');
      window.location.href = '/login';
    }
    return Promise.reject(error);
  }
);
```

### Q2: 如何处理版本 Logo？

```javascript
const getLogoUrl = (logo) => {
  if (!logo) {
    return '/static/icons/version_logos/default.png';
  }
  if (logo.startsWith('/')) {
    return `http://localhost:8080${logo}`;
  }
  return logo;
};
```

### Q3: 版本 ID 是数字还是字符串？

- `version.versionId` 是 **数字** 类型
- `card.versionId` 也是 **数字** 类型
- `user.id` 是 **字符串** 类型
