## 📋 前端对接配置

### 正确的 .env 配置

```env
VITE_DEV_PORT=5173
VITE_API_HOST=localhost
VITE_API_PORT=8080
VITE_API_PREFIX=/v1/api
VITE_API_BASE_URL=http://localhost:8080
```

**注意**：`VITE_API_BASE_URL` 只设置到端口，后面加 `/v1/api` 前缀

### API URL 拼接规则

```javascript
// 健康检查（不走 prefix）
const healthUrl = `${VITE_API_BASE_URL}/health`;
// → http://localhost:8080/health

// 其他 API（走 prefix）
const loginUrl = `${VITE_API_BASE_URL}${VITE_API_PREFIX}/auth/login`;
// → http://localhost:8080/v1/api/auth/login
```
