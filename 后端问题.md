### 接口 3：创建/更新版本配置（upsert）

```
PUT /v1/api/admin/version-config/:versionId
Authorization: Bearer <admin_token>
```

**请求体**：
```json
{
  "rarities": ["QCSER", "PSER", "SER", "UR", "SR"],
  "grades": [
    { "name": "99品", "desc": "完美全新", "hasSub": false },
    { "name": "9品", "desc": "近新优品", "hasSub": false }
  ],
  "defaultRarity": "QCSER",
  "defaultGrade": "9品"
}
```

**校验**：
- `defaultRarity` 必须在 `rarities` 中
- `defaultGrade` 必须在 `grades[].name` 中
- `rarities` 至少 1 项

**成功 200**：返回完整的 versionConfig 对象（含 updatedAt）

## ✅ 中文编码乱码问题（已修复）

**修复时间**：2026-06-20

### 修复内容

1. **Express 响应头**：添加 UTF-8 charset
2. **body-parser**：指定 `charset: 'utf-8'`
3. **数据库**：清理了乱码数据

### 代码修改

```javascript
// src/index.js
app.use((req, res, next) => {
  res.setHeader('Content-Type', 'application/json; charset=utf-8');
  next();
});

app.use(express.json({ charset: 'utf-8' }));
app.use(express.urlencoded({ extended: true, charset: 'utf-8' }));
```
